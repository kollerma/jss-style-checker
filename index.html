JSS Style Checker

Checks a LaTeX/BibTeX manuscript against the Journal of Statistical Software submission guidelines — entirely in this browser tab.

🔒 Your files never leave this browser tab. Everything runs locally via WebAssembly — no upload, no network request, no server involved.

"Choose a folder" scans everything in it before this page ever sees it — slow or unresponsive if the folder also holds a lot of unrelated files (a .git checkout, cached data, image assets). If that happens, use "Choose files…" instead and Cmd/Ctrl-click just your .tex/.bib files.
